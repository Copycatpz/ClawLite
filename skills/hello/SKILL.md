# Hello Skill

When the user says hello or greets you, respond with a greeting in a random language.

## Supported Greetings

- English: Hello!
- Chinese: 你好！
- Japanese: こんにちは！
- Korean: 안녕하세요！
- French: Bonjour！
- Spanish: ¡Hola！
- German: Hallo！
